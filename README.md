# yt-videos

Текстовые конспекты видео: транскрибации, переработанные в удобные для чтения статьи —
с оглавлением, структурой, выделенными ключевыми мыслями и адаптивной вёрсткой
(мобильный + десктоп, светлая/тёмная тема).

🔗 **Сайт:** https://roman-purtow.ru/yt-videos/

## Структура

```
yt-videos/
├── index.html                      # хаб со списком всех конспектов
├── .nojekyll                       # отключает обработку Jekyll на GitHub Pages
├── 2026-06-13-marketolog-300k/
│   ├── index.html                  # конспект-статья
│   └── source.md                   # исходная транскрибация видео
├── 2026-06-13-yandex-direct/
│   ├── index.html                  # конспект-статья
│   └── source.md                   # исходная транскрибация видео
└── 2026-06-08-getcourse/
    ├── index.html                  # конспект-статья
    └── source.md                   # исходная транскрибация видео
```

## Как добавить новый конспект

1. Создать папку вида `ГГГГ-ММ-ДД-тема/`.
2. Положить туда исходную транскрибацию (`.md`) и собрать статью `index.html`
   (можно взять за основу любой существующий конспект — стили инлайнятся в один файл,
   внешних зависимостей нет).
3. Добавить карточку-ссылку в корневой `index.html` и строку в таблицу выше.

Все страницы — чистая статика: ни сборки, ни npm, ни внешних CDN.
